## 2. 实现方法

### 2.1 总体方案
1. 首先创建 `schema_migrations` 表（version, name, applied_at）
2. 每个迁移用事务包裹：执行 SQL → 记录版本号 → 提交
3. 启动时检查已应用的版本，跳过已执行的迁移
4. 提供 `CurrentVersion()` 方法查询当前版本
